# 霞ヶ原駐屯地 複数ページ版

スクロール型ではなく、ページごとに分かれた公式サイト形式です。

## ページ

- `index.html` ホーム
- `about.html` 駐屯地について
- `units.html` 部隊紹介
- `development.html` 開発状況
- `announcements.html` アナウンス
- `topics.html` トピック

## 編集

内容は基本的に `data.js` だけ編集します。

画像は `images` フォルダへ入れてください。

## GitHub Pages

リポジトリのルートへ、このフォルダ内のファイルをすべてアップロードしてください。
その後 `Settings > Pages > Deploy from a branch > main / root` を選択します。
